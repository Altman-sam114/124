# v3.7-preflight.38 剩余 UI 文案抽样收口记录

完成日期：2026-07-05

## 1. 本轮目标

在 v3.7-preflight.37 收口 App/UI 边界文案之后，继续抽样处理未完全覆盖的 UI 面板、开局引导、HUD、经济、外交、州郡经营、总管和单位提示中的玩家可见工程感文案。

## 2. 范围

- `FirstTurnGuideView`：开局引导中的 AI / Xcode / 发布验证可见词和 scenario fallback。
- `HUDView`、`AppContainer`、`ReleaseChecklistView`：顶部标题、交互日志和战局复核里的 scenario fallback，不再直出 raw id、草稿或旧版口径。
- `EconomyPanelView`、`EventLogView`：面板摘要中的工程式分隔符。
- `DiplomacyPanelView`、`RegionInspectorView`、`GeneralCommandPanelView`、`UnitTooltipView`：常见斜线分隔和 VoiceOver 读法。
- `AgentPanelView`：隐藏 raw JSON 原文，净化 context / command result / error 文案。
- `GeneralProfileView`：将领画像 initials 不再 fallback 到 raw 英文 id。
- `GameSettingsView`、`ReleaseCandidateMenu`：设置和复核入口改为玩家口径。
- 文档：`README.md`、`update_log.md`、`md/flow/flow.md`、`md/flow/flowchart.md`、`md/plan/plan.md`。

## 3. 关键改动

- `FirstTurnGuideView` 的“朝堂和 AI 行动”改为“朝堂自动行动”，“军令校验和规则引擎”改为“军令规则判定”，发布/构建/烟测提示改为战局复核提示。
- `FirstTurnGuideView`、`HUDView`、`AppContainer`、`ReleaseChecklistView` 的旧剧本 / 草稿 fallback 改为“当前战局”或“自定战局”，不再直出 raw id、草稿或旧版口径。
- `ReleaseCandidateMenu` / `ReleaseChecklistView` 的“发布前检查 / 发布候选检查”改为“战局复核 / 当前战局”，发布测试词改为实机复核口径。
- `AgentPanelView` 不再展开显示 `rawJSON`，`contextSummary`、`result.message`、`result.errors` 统一经过展示净化。
- `EconomyPanelView` 的征发耗费说明、`EventLogView` 的方面军令统计、`DiplomacyPanelView` 的势力/盟从、`RegionInspectorView` 的太守政策说明、`GeneralCommandPanelView` 的官阶/风格说明改用中文标点，不再使用斜线或竖线工程分隔。
- `RegionInspectorView`、`UnitInspectorView` 的列表摘要改用中文顿号分隔。
- `UnitTooltipView` 的 VoiceOver 兵力读法改为“兵力 N，上限 M”。
- `GeneralProfileView` 的将领画像 fallback 改为“将 / 将领”，不再从 raw 英文 id 生成缩写。

## 4. 并发扫描

本轮启动两个只读子 agent：

- UI 入口/设置子 agent：扫描 `FirstTurnGuideView`、`GameSettingsView`、`ReleaseCandidateMenu`、`HUDView`、`RootGameView`、`NewGameButton`、`InfoPanelToggle`、`EconomyPanelView`。
- 战报/地图子 agent：扫描 `EventLogView`、`AgentPanelView`、`GeneralCommandPanelView`、`GeneralProfileView`、`UnitTooltipView` 和 SpriteKit 地图节点。

主线程同步执行 `rg` 可见字符串抽样扫描，并优先处理明确低风险候选。两个子 agent 后续返回的 raw JSON、context/result 原文、发布流程词、scenario fallback 和画像 initials 风险也已并入本轮修复。本记录仍不声称完成全 UI 穷尽审计。

## 5. 架构边界

- 不改 enum rawValue 或内部审计 id；只处理玩家可见文本和辅助读法。
