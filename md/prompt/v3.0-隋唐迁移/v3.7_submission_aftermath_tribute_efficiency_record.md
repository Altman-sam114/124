# v3.7-preflight.102 归附善后贡赋效率落地记录

完成日期：2026-07-07

## 目标

从总提示词 §0.2 “完整忠诚、叛乱、贡赋、俘虏、安置等归附善后实际规则”中继续切出低风险子问题：让 `.101` 已落地的治安/顺从压力影响后续府库收入，形成“贡赋效率”实际规则。

## 实现

- `EconomyRules.income(for:map:)` 聚合受控州郡收入时读取 `RegionNode.occupationState`。
- 新增局部贡赋效率计算：
  - 无 `occupationState` 时效率为 `1.0`。
  - 抵抗每点带来小额折减。
  - 顺从超过 50 后提供小额恢复，但不超过基础产出。
  - 效率下限保留为 `0.55`，避免归附州郡经济完全归零。
- `.101` 的归附善后压力提高抵抗、降低顺从后，会压低后续回合丁口、军械、粮草收入。
- 既有 `Command.governRegion(.pacifyPopulation)` 安民降低抵抗、提高顺从后，会自然恢复贡赋效率。
