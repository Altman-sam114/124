完成日期：2026-07-06

## 目标

继续推进完整 v3.7 发布候选前置收口，处理自动回合、元帅配置、legacy 映射失败和元帅解析/编译失败路径中可能进入玩家视野的旧二战人物名、英文性格描述、工程词和底层异常文案。

本轮使用两个并发子 Agent 做只读扫描：

- 子 Agent Poincare：定位自动回合异常和元帅诊断兜底，建议收口 `TurnManager` 与 `ZoneCommanderAgent`，保留 raw JSON / prompt / schema 内部合同。
- 子 Agent Wegener：复扫下一批低风险可见英文，提出 `DataLoader` 初始日志、MapEditor 导出标题和 dataNotes 作为后续候选，不纳入本轮改动。

- `MarshalAgentConfig.automatic` 的 legacy 元帅展示名改为“伦德施泰特 / 艾森豪威尔”，并把旧剧本与隋唐势力的 personality 文案改为中文。

- `TurnManager` 的 legacy Agent D 映射失败路径改用 `userFacingError(_:)` 包装底层错误。
- `TurnManager.userFacingDiagnostic(_:)` 补齐古德里安、元帅、行军总管、君主、schema、provider、local-model、fallback、agent、directive、diagnostic、breakthrough 等常见诊断词替换。
