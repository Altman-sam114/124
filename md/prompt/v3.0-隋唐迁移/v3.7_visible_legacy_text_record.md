# v3.7-preflight.31 玩家可见旧英文兜底收口记录

完成日期：2026-07-05

## 目标

继续压低玩家可见路径中的二战/英文残留，把旧剧本 fallback、军队详情和经济战报里仍可能露出的英文兜底改为中文口径。

## 实现范围

- `Faction.displayName` 的 legacy 势力从 `Germany` / `Allies` 改为 `德军（旧）` / `盟军（旧）`。
- `GamePhase.displayName` 的 legacy 阶段从“AI/玩家行动（旧剧本）”改为通用中文阶段名。
- `VictoryReason.displayName` 的 legacy 胜负原因从德军/盟军/巴斯托涅等旧题材名改为泛化旧剧本中文兜底。
- `HexDirection` 新增中文 `displayName`。
- `UnitInspectorView` 的方向集合显示从 E / NE / NW / W / SW / SE 改为中文方向。
- `EconomyRules` 的府库不足、征发入队、府库结算、自动补员、补给完成、部署完成和无安全后方地块事件日志中文化。

## 关键边界

- 不改 `Faction`、`GamePhase`、`HexDirection`、`VictoryReason` 或其他 enum rawValue。
- 不改存档 JSON schema、数据文件、规则数值、AI 决策、命令管线或地图数据。
- 不删除 legacy 阿登 fallback、旧测试语义或内部类型名。
- 不执行 Xcode build、XCTest、模拟器、app 启动或 UI 烟测。
