# v3.7-preflight.62 legacy 总管配置中文兜底

完成日期：2026-07-06

## 目标

在 AI 诊断净化口径对齐之后，继续处理 legacy `general_agents.json` 中 `guderian` 配置优先于代码 fallback 加载时，英文展示名和英文 personality prompt 仍可能进入旧 LocalLLM / legacy Agent D 上下文的问题。

## 实现范围

- `WWIIHexV0/Data/general_agents.json` 中 legacy `guderian` 的展示字段中文化：
  - `name` 改为“古德里安”。
  - `personalityPrompt` 改为中文装甲突破、道路机动、集中兵力和快速合围偏好。
- `GameAgent(definition:)` 保留从 `general_agents.json` 读取 id、faction、role、assignedDivisionIds 和 command style 数值判断。
- 对 `definition.id == "guderian"` 的 legacy 配置做代码层中文兜底：
  - `name` 固定为“古德里安”。
  - `personality.prompt` 固定为中文作战偏好。
  - `traits` 中的 `breakthrough` 显示为“突破”。
- `guderianFallback` 的 traits 同步改为“突破”，保持读取 JSON 成功与失败时的展示口径一致。

## 边界

- 不改 JSON schema、id、rawValue、command style、辖下单位 id 或 Codable 结构。
- 不改 `AgentPromptBuilder`、`LocalLLMDecisionProvider`、`AgentDecisionParser` 或命令解析合同。
- 不改 AI 决策策略、命令结构、规则执行、存档字段或运行时状态。
- `AgentPromptBuilder` 的 raw id / 英文 type 诱导风险保留为后续独立小片。
