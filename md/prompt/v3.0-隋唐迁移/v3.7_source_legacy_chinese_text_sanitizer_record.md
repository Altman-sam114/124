# v3.7-preflight.83 源头 legacy 中文势力/国家/地名展示净化记录

完成日期：2026-07-06

## 目标

在 v3.7 发布候选前置收口中，继续降低玩家可见路径出现 legacy 二战中文势力、旧国家、旧地名、旧单位词和英文角色/工程词的风险。

## 范围

- `TurnManager` 自动回合失败、方面军令诊断和错误兜底展示。
- `DiplomacyState` 外交事件、归附交接、善后压力、善后处置和国家摘要。
- `GameLogEntry` 源头战报消息净化。

## 实现

- `TurnManager.userFacingDiagnostic(_:)` 补齐 `德军 / 盟军 / 旧剧本德方 / 阿登 / 巴斯托涅 / 装甲 / Germany / Allies / Guderian / Field Marshal / rawJSON / schema / provider` 等展示替换。
- `DiplomacyState` 复用 `displayFactionName(_:)` 生成外交事件、交接和善后摘要，并让 `displayCountryName(_:)` 净化旧中文国家名。
- `GameLogEntry.sanitizedMessage(_:)` 补齐英文角色、英文势力和工程词净化，防止源头战报绕过 UI 层 helper。

## 边界

- 不改 `Faction` rawValue、legacy case、JSON id、country id、record id 或 Codable schema。
- 不改 `Command` / `ZoneDirective` / `WarCommandExecutor` / `RuleEngine` 管线。
- 不改 `Division.name`、`RegionNode.name`、`CountryProfile.name` 的存储值。
- 不新增规则效果，不改外交判定、归附交接、善后治理或存档格式。
