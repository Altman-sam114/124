# v3.7-preflight.51 MapEditor 与主游戏 raw id 可见文案复扫收口记录

完成日期：2026-07-06

## 目标

本轮在 v3.7-preflight.50 之后继续做发布候选前置补洞，聚焦 MapEditor 右侧信息面板、状态反馈、导出错误，以及主游戏州郡/军队详情、选择日志、战报/朝堂/外交记录文本中仍可能进入玩家视野的 raw 坐标、文件名、raw id 和工程词。

## 范围

- MapEditor 底图、信息、地点、状态和导出错误文案。
- MapEditor region / theater 名称缺省 fallback。
- 主游戏交互日志、州郡详情、军队详情中坐标直出问题。
- 主游戏方面 / 行军防区展示名的 raw id fallback。
- Agent、战报和外交记录摘要中的 `region_*`、`theater_*`、`front_zone_*`、`obj_*`、`agent_*` 等通用 raw id 净化。

## 非目标

- 不改变 JSON schema、资源文件名、导出文件名、id、rawValue、objectiveId、record id 或兼容字段。
- 不改变 MapEditor 文档结构、默认资源桥、主游戏加载顺序、AI 决策、命令结构、RuleEngine 或任何规则数值。
- 不运行 Xcode build/test、XCTest、模拟器、Probe、Smoke、Stage Regression、Dynamic Theater Regression、Full、性能测试或 app 启动。

## 并发扫描

本轮按人工要求使用只读子 agent 加速：

- MapEditor 方向：扫描 `MapEditorView`、`MapEditorViewModel`、`MapEditorDocument`、`MapEditorExporter`、`MapEditorGameResourceBridge`，定位坐标、文件名、raw objective id、raw fallback 名称和导出错误残留。
- 主游戏方向：扫描 `AppContainer`、`RegionInspectorView`、`UnitInspectorView`、`MapDisplayAdapter`、`AgentPanelView`、`EventLogView`、`DiplomacyPanelView`，定位坐标直出、方面/防区 raw name 透传和记录摘要 raw id 净化缺口。

子 agent 均只读，未编辑文件，未运行重测试。

## 关键变更

- `MapEditorView` 将“游戏资源”改为“战役资料”，底图偏移输入从 `X/Y` 改为“横向偏移/纵向偏移”，底图文件名展示改为“底图：已导入”，坐标展示改为“位置：第 X 列，第 Y 行”，“胜负点代号”改为“胜负地点标记”。
- `MapEditorViewModel` 的新建、选择、导入、保存、读取、预览、导出和扩展反馈去掉 raw 坐标、文件名、目录名和“内存”工程词；自动城池/关隘名改为“未命名城池/未命名关隘”。
- `MapEditorDocument` 的州郡/方面缺省名不再回落到 `id.rawValue`，改为“未命名州郡/未命名方面”。
- `MapEditorExporter` 的未分配州郡错误、自动目标名和自动粮仓名不再暴露 raw 坐标或 raw objective id。
- `MapEditorGameResourceBridge` 的未知地形错误不再拼接 raw terrain。
- `AppContainer`、`RegionInspectorView`、`UnitInspectorView` 不再向玩家直出 `q,r` 坐标。
- `MapDisplayAdapter` 对方面和行军防区展示名做 raw id fallback，遇到空值、等于 rawValue、下划线或 `theater/front` 前缀时改用“当前方面/行军防区”。
- `AgentPanelView`、`EventLogView`、`DiplomacyPanelView` 增加通用 raw id 正则净化，覆盖 `region_*`、`theater_*`、`front_zone_*`、`obj_*` 和 `agent_*`。

## 遗留风险

- 本轮没有真实启动 MapEditor 或主游戏确认界面视觉效果。
- 内部 id、schema、rawValue、资源文件名、objectiveId 和 record id 仍按兼容合同保留英文或 ASCII。
- 完整 v3.7 发布候选仍需要授权构建、启动、MapEditor 操作、主游戏点击、AI 多回合、界面烟测和云端结果包验收。
