# v3.7-preflight.101 归附善后治安压力落地记录

完成日期：2026-07-07

## 目标

从总提示词 §0.2 “忠诚、叛乱、贡赋、俘虏、安置等归附善后实际规则”中切出低风险子问题：归附实体交接后，既有 `SubmissionAftermathRecord.riskLevel` 不再只作为复盘提示，而是对受影响州郡写入实际治安/顺从压力。

## 实现

- `CommandExecutor.executeSubmissionHandoff` 在生成 `SubmissionAftermathRecord` 后调用 `applySubmissionAftermathPressure(from:in:)`。
- `applySubmissionAftermathPressure` 遍历 `affectedRegionIds`，读取或创建 `OccupationState`，按风险等级提高 `resistance`、降低 `compliance`。
- 风险等级映射：
  - `low`：治安压力 +4，顺从 -4。
  - `guarded`：治安压力 +8，顺从 -8。
  - `high`：治安压力 +12，顺从 -12。
- 落地后追加外交战报，提示哪些州郡治安承压，需要太守安民或整修支撑。
- `DiplomacyState` 的善后与治理边界说明同步更新，明确安民等既有州郡经营可调整治安/顺从，但不触发额外忠诚、叛乱、贡赋、俘虏、安置系统。
