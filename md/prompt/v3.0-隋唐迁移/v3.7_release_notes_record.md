# v3.7-preflight.8 发布说明与资产边界记录

完成日期：2026-07-05

## 1. 本轮定位

本轮继续补 v3.7 发布候选前置的“发布说明 / 资产边界 / 验证口径”缺口，目标是让发布前检查面板和文档明确说明当前代码已经接入什么、哪些地图标识仍是最小代码绘制、哪些运行时风险仍需人工授权重测。

本轮不引入外部素材，不新增 asset catalog，不改变地图、存档、命令、AI、胜负、补给、移动或战斗规则。

## 2. 实现摘要

`ReleaseChecklistView` 更新为 `天命开唐 Agent · v3.7-preflight.8`，并新增：

- “发布说明”：说明首发定位、当前入口和验证口径。
- “资产边界”：说明城池、关隘、粮仓、渡口、港口、AI 箭头和前线墨线当前都是代码绘制或派生显示。

- “仍待确认”把正式资产待办收敛为运行时截图检查和正式资产替换决策。
