# v3.7-preflight.74 将领与总管面板 legacy 文案复扫收口记录

## 目标

收口将领档案和总管军令面板中 legacy 将领名、旧军阶、旧地名、旧兵种词和 raw id 的玩家可见直出风险。

## 范围

本轮只改 `GeneralProfileView` / `GeneralCommandPanelView` 的展示 helper 和文档；不改 `GeneralData`、`GeneralAssignment`、`FrontZone`、`Division.name`、`PlayerPlannedOperation`、JSON、rawValue、存档、AI 决策、命令结构、`RuleEngine` 或运行时状态。

## 实现摘要

- `GeneralProfileView` 的将领名、军衔、履历、所属防区、所属军队和画像 accessibility 文案进入局部展示净化。
- `GeneralCommandPanelView` 的将领名、军衔、履历、防区名、所属军队、目标州郡、预备军令目标和头像 / 查看档案 accessibility 文案进入局部展示净化。
- 展示 helper 清理 `general_*`、`agent_*`、`commander_*`、`front_zone_*`、`theater_*`、`region_*`、`objective_*`、`division_*`、`unit_*` 等常见 raw id。
- 展示词表替换 `Heinz Guderian`、`Guderian`、`古德里安`、`Panzer`、`armor`、`breakthrough`、`Field Marshal`、`Army Commander`、`General`、`Ardennes`、`Bastogne`、`St. Vith`、`St Vith`、`Sedan`、`阿登`、`巴斯托涅`、`圣维特`、`德军`、`盟军` 和旧兵种词。

## 并发扫描结论
