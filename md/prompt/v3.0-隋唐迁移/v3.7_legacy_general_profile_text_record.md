# v3.7-preflight.46 Legacy 将领档案可见文本收口记录

完成日期：2026-07-06

## 目标

继续推进 v3.7 发布候选前置文案收口，处理 legacy 将领数据中仍会被 `GeneralCommandPanelView` 和 `GeneralProfileView` 直接展示的英文军衔与履历，并补齐将领技能与总管展示名的中文显示兜底。

本轮只清理展示字段和展示映射，不改将领 id、英文正名、localizedName、技能 raw id、倾向、归属、画像、初始分配或 agent prompt。

## 改动范围

- `WWIIHexV0/Data/generals.json`
  - `rank` 改为中文军衔。
  - `biography` 改为中文旧剧本履历说明。
- `WWIIHexV0/Agents/GeneralRegistry.swift`
  - `GeneralData.commanderConfig` 的总管展示名优先使用 `localizedName`，空值才 fallback 到 `name`。
- `WWIIHexV0/UI/GeneralCommandPanelView.swift`
  - 补齐隋唐与 legacy 将领 `skills` raw id 的中文显示映射。
- `WWIIHexV0/UI/GeneralProfileView.swift`
  - 复用同一套技能显示映射，避免将领档案继续显示“未知军略”。

## 保留边界

- 保留 `id`、`name`、`localizedName`、`faction`、`commandStyle`、`skills`、`portrait`、`preferredTheaterIds`、`preferredRegionIds`、`baseLoyalty`、`baseSatisfaction`。
- 保留英文 `name` 字段；总管展示名只在 Swift 侧优先读 `localizedName`。
- 保留 `skills` raw 值，只补 Swift 显示映射，raw 值仍是数据合同。
- 不修改 `general_agents.json` 的 `personalityPrompt`、`commandStyle`、`role`、`assignedDivisionIds` 或 agent 名称。
- 不修改 `suitang_generals.json`；其 `rank` 和 `biography` 已是中文，拼音 `name` 作为数据正名暂不纳入本轮。

## 轻量检查要求

- `jq empty WWIIHexV0/Data/generals.json`
- `swiftc -parse WWIIHexV0/Agents/GeneralRegistry.swift WWIIHexV0/UI/GeneralCommandPanelView.swift WWIIHexV0/UI/GeneralProfileView.swift`
- `git diff --check`
- 本轮改动文件尾随空白扫描。
- 本轮改动文件冲突标记扫描。
- 定向残留扫描：`generals.json` 的 `rank`、`biography`、`localizedName` 不应残留英文字母；将领面板不应继续包含“未知军略”。

## 未执行

按 `md/test/test.md` 当前规则，本轮不跑 Xcode build、XCTest、模拟器、Probe、Smoke、Stage Regression、Dynamic Theater Regression、Full、性能测试或 app 启动。将领面板真实展示效果仍需后续授权运行时验证或云端结果包复核。

## 遗留风险

- legacy 将领 `name` 仍保留英文正名，服务 AI/审计兼容链路。
- `general_agents.json` 的 `personalityPrompt` 仍是英文内部 prompt，后续如要迁移，需要单独评估 AI 行为影响。
- `AgentConfiguration.guderianFallback` 仍含英文 fallback 名称和 prompt；该路径仅在 JSON agent 配置加载失败时使用，后续可单独做 fallback 可见文本和内部 prompt 切片。
